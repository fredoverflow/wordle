## Go

- Install Go from https://go.dev/dl
- Download `go.mod` and `wordle.go` (and the `.txt` files)

### Git Bash / PowerShell

```
cd Downloads
go build
./wordle
```

### CMD

```
cd Downloads
go build
wordle
```

## Java

- Install Java from https://adoptium.net
- Download `Wordle3.java` (and the `.txt` files)

```
cd Downloads
javac Wordle3.java
java Wordle
```
